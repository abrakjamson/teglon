# Teglon

In the years when the old monasteries still kept their own counsel, there were tales of a carved practice passed among the elders of the house. The monks, so the stories say, did not speak of age as a matter of decline; they spoke of it as a kind of sharpening. They slept little, ate sparingly, and spent long hours arranging small geometric pieces upon stone tables, each one a fragment of a larger pattern. Their legends claim that those who studied the arrangement with sufficient devotion could live well beyond a hundred years, not because the work was magical, but because it taught the mind to remain steady.

Teglon is said to be the digital echo of those tiles, preserved in a form that can be carried into the modern world. It is a quiet puzzle of placement and rotation, where each piece must find its chamber and each completed arrangement must yield a single unbroken line. What remains of the monastery in this version is not incense or candlelight, but the discipline of the hands and the hush of concentration.

## What you will find here

- A browser-based puzzle game with multiple exercises
- Drag-and-drop tile placement with rotation controls
- Hint, reset, and sound options
- A simple, calm interface built in the spirit of contemplative design

## Install and run

1. Install the project dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the local Vite address shown in the terminal, usually at http://localhost:5173.

## Build for production

To create a production build:

```bash
npm run build
```

To preview the build locally:

```bash
npm run preview
```
